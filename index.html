How to validate a Credit Card number?

The Luhn formula is a simple algorithm used to validate credit card numbers.

How Luhn Algorithm works?

  • Drop the last digit from the number. It is the control number.
  • Starting from right double every odd number (1, 3, 5...).
  • If multiplied number is greater than 9, subtract 9 from it.
  • Add all the numbers together.
  • Add to the sum the control number
  • If the final sum is a multiple of 10 (ends with 0), then provided number is considered valid according to the Luhn formula.
